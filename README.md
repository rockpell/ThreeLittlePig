# ThreeLittlePig

늑대를 피해 살아남는 돼지의 생존 게임입니다.

많은 행동을 할 수록 더 많은 점수를 얻을 수 있습니다.

이동: 방향키 또는 wasd

벽 설치: 마우스 왼쪽 클릭

설치할 벽 종류 변경: 키보드123, 아이콘 클릭, 마우스 휠 조작

벽 부수기: 마우스 오른쪽 클릭

늑대 옷입히기: 마우스 오른쪽클릭

벽에 불 붙히기: F
